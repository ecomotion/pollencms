Structure des répertoires

Mat simple cms est divisé en 3 répertoires principaux:

  • core
  • vendors
  • site

Le core

Le répertoire core, comme son nom l'indique contient le coeur du CMS. Il est divisé en sous répertoires :
  • admin: contenant les fichiers php de l'interface d'administration
  • jscripts: contenant les fichiers jscripts de l'interface d'admin
  • locale: contenant les fichiers de traduction de l'interface d'admin
  • lib: contenant les classes utilisées par le système : PPage, PFile ....

Vendors

Le répertoire vendors contient les librairies externes utilisées par le cms, fckeditor, phpmail, smarty template engine ....

Ainsi il est facile de mettre à jour le cms lors d'une nouvelle version de l'un de ces outils sans risquer de faire bugger l'ensemble du système.

Site

Le répertoire site contient tout ce qui permet de personnaliser son site. C'est ce répertoire qu'il vous faudra modifier pour personnaliser votre site.
Le répertoire site est divisé en sous répertoire:
  • config contenant les fichiers de configuration du site
  • pages contenant les pages du site ansi que les fichiers cache et les fichiers uploadés (images, vidéos ....)
  • templates contenant les gabarits du site ainsi que les fichiers de style css, javascripts propre au site, images ....
