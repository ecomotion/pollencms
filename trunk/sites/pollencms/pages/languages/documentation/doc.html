Un système sans base de données

Contrairement à beaucoup d'autres CMS, ce système se différencie par le fait qu'il n'utilise pas de base de données.
Les données sont naturellement stockées dans des fichiers, en fait à chaque page correspond un fichier html de contenu.

Gestion des gabarits : Smarty template engine

Mat simple CMS repose sur le système de template smarty. En fait à chaque page on peut associer un gabarit smarty ou lui associer le gabarit par défaut.
Smarty est un système de gabarit très performant gérant efficacement le cache des pages. Ainsi mat simple cms bénéficie automatiquement de ce système de cache.

Une page = un fichier de configuration

Pour enregistrer les informations de configuration associées à une page comme le gabarit, le titre de la page, le style .... matsimplecms associe à chaque page
un fichier de configuration (un fichier .ini). Ainsi si je crée la page accueil.html, matsimplecms créera automatiquement un fichier accueil.ini dans lequel il
stockera les configurations associées à cette page.dfsfdsfds
